## About

This is the mobile application component to the MDMS Generic Device Sign-out Service. This React Native application allows team members to sign out a shared mobile device.

Additional component can be fond in these repos:

[MDMS Generic Device Sign-out API](https://github.com/bcdevops/mdms-device-signout-api)

[MDMS Generic Device Sign-out Mobile Application](https://github.com/bcdevops/mdms-device-signout-app)

## Getting Help or Reporting an Issue

Send a note to bcdevexchange@gov.bc.ca and you'll get routed to the right person to help you out.
